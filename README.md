# Tarea 3: Redes Neuronales Convolucionales (ConvNets) 🧠👟

Este proyecto explora el uso de Deep Learning para la clasificación de imágenes. 

## Contenido:
* **Análisis de Kernels:** Experimentamos con filtros de 3x3 hasta 6x6.
* **Prueba Real:** Evaluamos el modelo con mi propia caligrafía (83.33% de precisión).
* **Aplicación Nube Andina:** Reflexión sobre el control de calidad en calzado de fibra de alpaca.
